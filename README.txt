This is a React project that calls an API for a deck of cards. It demonstrates how to consume APIs using axios framework.

  REQUIREMENTS

 1)for building and running this application you need to have Nodejs,NPM(v8) and VScode(latest version)installed in your system.
 
 2)Clone this project: https://github.com/Abelinho/ReactCardAPI.git, import it into your vscode, cd to the location where the project is and run the command: npm start
